# AUC

A dead-simple auto-clicker in C99. This project uses libuiohook for I/O hooks.

## BUILD INSTRUCTIONS

Clone this repository
```
   %  git clone https://github.com/vs-123/auc.git
```

Make a `build/` directory inside the project root and use CMake to build
```
   %  cd auc
   %  mkdir build
   %  cmake ..
   %  cmake --build .
```

You may execute the generated binary `auc`
```
   %  ./auc
   #########################
   #  AUC -- AUTO-CLICKER  #
   #########################
   
   * AUTHOR: vs-123 @ https://github.com/vs-123
   * PRESS F6 TO TOGGLE.
```
